# PWA Icons

This directory should contain the following icon files for PWA functionality:

## Required Icons

- `icon-192.png` - 192x192 pixels, PNG format
- `icon-512.png` - 512x512 pixels, PNG format
- `apple-touch-icon.png` - 180x180 pixels, PNG format (optional, falls back to icon-192.png)
- `favicon.ico` - 32x32 pixels, ICO format (optional)

## Icon Design Guidelines

- Use a simple, recognizable design that works at small sizes
- Use the primary brand color (#0ea5e9) as the background
- Include a smartphone or code symbol
- Ensure good contrast for visibility
- Test icons at all sizes to ensure readability

## Generating Icons

You can use tools like:
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [Favicon.io](https://favicon.io/)

Or create them manually using design tools like Figma, Photoshop, or GIMP.

## Temporary Solution

For development, you can use placeholder icons. The app will work without them, but users won't see a proper icon when installing the PWA.
